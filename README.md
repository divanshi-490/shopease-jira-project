## shopease-jira-project
## Project Description
This project is based on managing an Online Shopping system (ShopEase) using Jira as a project management tool. It focuses on understanding how projects are planned, organized, and tracked in a structured way.
## Features
Creation of Epics, Tasks, and Sub-tasks  
Workflow management (To Do, In Progress, Done)  
Task assignment and tracking  
Bug tracking and issue management  
Comments for communication  
Time tracking (Log work feature)  
Project dashboard overview  
## Tools & Technologies Used
Jira (Project Management Tool)
## Project Structure
 Epics: User Management, Product Management, Cart & Wishlist  
Tasks and Sub-tasks under each Epic  
Workflow tracking using Jira board  
## Screenshots
Screenshots of Jira board, Epics, Tasks, Workflow, and Time Tracking are included in the repository.
## Learning Outcomes
Understanding of project management concepts  
Knowledge of task organization and workflow tracking  
Experience with real-world project management tool (Jira)  
## Conclusion
This project provides a basic understanding of managing software projects using Jira. It demonstrates how tasks are organized, tracked, and completed in a structured workflow.
